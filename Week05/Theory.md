```c++
for (int i = 0; i <= 20; i *= 2, std::cout << ++i << " " << std::endl);
```

**Задача 2:** Напишете еквивалентен код на следния програмен фрагмент, като използвате само оператор switch:

```c++

if (a == 7) std::cout << "seven";
if ( (a == 3) || (a == 5) ) std::cout << "three and five";
```
